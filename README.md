> Overview

Platform Surveillance System is a Python-based automation tool that monitors system performance and generates periodic log reports. It collects real-time data on CPU usage, memory utilization, disk usage, network activity, and running processes, and stores the information in timestamped log files for analysis and monitoring.

> Features

Automated system monitoring

CPU and RAM usage tracking

Disk partition usage report

Network I/O statistics

Detailed running process report (PID, name, user, status, CPU %, memory %)

Timestamp-based log file generation

Periodic execution using scheduler

Command-line argument support
